# 📊 Student Grade Calculator

A clean, interactive web-based grade calculator for students.  
Enter your subject names, marks obtained, and total marks — get your **percentage**, **CGPA**, and **grade** instantly.

## ✨ Features
- Add / remove any number of subjects
- Calculates percentage, CGPA (out of 10), and letter grade
- Visual progress bar for each subject
- Grading scale: O / A+ / A / B+ / B / C / F
- Fully responsive — works on mobile too
- No installation needed — just open `index.html` in a browser

## 🚀 How to Run
1. Download or clone this repository
2. Open `index.html` in any web browser
3. That's it! No server or setup required.

```bash
git clone https://github.com/YOUR_USERNAME/grade-calculator.git
cd grade-calculator
# Open index.html in your browser
```

## 🎓 Grading Scale

| Percentage | Grade | Grade Points |
|------------|-------|--------------|
| 91 – 100   | O     | 10           |
| 81 – 90    | A++   | 9.5          |
| 71 – 80    | A+    | 9            |
| 61 – 70    | A     | 8            |
| 51 – 60    | B++   | 7            |
| 41 – 50    | B+    | 6            |
| 35 – 40    | B     | 5            |
| 25 – 34    | C     | 4            |
| Below 25   | F     | 0            |
